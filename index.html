Differential Equation Checker

Instructions: Enter your differential equation components M(x, y) and N(x, y) in the fields below. The program will check if the equation is exact.

Rules for Input:

  • Use * for multiplication (e.g., enter "2*y" instead of "2y").
  • Use exp(x) for exponentials (e.g., enter "exp(2*y)" instead of "e^2y").
  • Use standard function notation (e.g., "sin(x)", "cos(x)", "tan(x)").
  • Use parentheses properly (e.g., "cos(x*y)").
  • Write powers explicitly using ^ (e.g., "y^2" instead of "y²").
